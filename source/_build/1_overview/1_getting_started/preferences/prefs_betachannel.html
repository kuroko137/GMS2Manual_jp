環境設定 - ベータ チャンネル


ベータ チャンネルの環境設定 ページでは、pGameMaker Studio 2 ベータ チャンネル へのオプトインをすることができます。ベータ チャンネルを利用することで、安定版 のアップデートだけではなく、より新しい ベータ版 がリリースされたときにも GameMaker Studio 2 をアップデートすることができるようになります。本製品におけるベータ版というのは、いわばテストが終わっていないリリース候補バージョンのようなものであり、プロジェクトへなんらかの影響を与えるバグが含まれている可能性があります。そのため、既存のプロジェクトで GameMaker Studio 2 Beta を使用し始める前に、ソース管理を使ってプロジェクトをバックアップし、別のコピーまたはブランチで作業を行うことをおすすめします。

ベータ チャンネルへオプトインする前に、エンド ユーザー利用許諾契約 (EULA) の更新 (主に Section 2) に同意する必要があります。次に、リスクを認識していること、そしてプロジェクトのバックアップが完了していることを確認する2つのチェックボックスにチェックを入れます。以上をよく確認したうえで "GameMaker Studio 2 Beta へ オプトイン" ボタンをクリックすると、GameMaker Studio 2 の再起動を求める画面が表示されます。再起動し終えると、ベータ チャンネルのフィードからアップデートを受け取れるようになります。このとき、すでにベータ版の IDE がリリースされている場合は、そのインストールが必要になります。

ベータ チャンネルへオプトインすると、環境設定 ボタンに "GameMaker Studio 2 Beta から オプトアウト" オプションが表示されるようになります。これをクリックすると、GameMaker Studio 2 の安定版をダウンロードするための リリース ノート が表示されるので、インストールしたいバージョン名をクリックしてください。:

なお、ベータ チャンネルは IDE とランタイムの 2つ のフィードから構成されています。ベータ版にオプトインすると、Beta IDE のインストールと同時に、ベータ版のランタイムもダウンロードしてインストールされます。しかし、使用しているランタイムは ランタイムの環境設定 からいつでも変更することができます。Master と Beta のランタイムを入れ替えて使用することも可能です。
